About CropSense

CropSense is an intelligent crop health advisory web app that allows farmers and gardeners to check crop issues based on symptoms or images. It supports multiple languages and provides multiple recommendations per crop-symptom combination. Future updates include AI-based disease detection, weather insights, and mobile app support.
